サンプル投稿: Sanityブログの使い方

公開日: 2024年1月15日

このSanityブログでは、以下のことができます：

  • リッチテキストエディタでの記事作成
  • 画像のアップロードと管理
  • 著者とカテゴリの管理
  • 投稿のスケジュール機能
  • Portable Textでの柔軟なコンテンツ構造

技術仕様

著者: 開発チーム
公開日: 2024年1月15日

このブログは以下の技術で構築されています：

  • CMS: Sanity.io
  • フロントエンド: Next.js 15
  • スタイリング: Tailwind CSS
  • 言語: TypeScript

コンテンツはSanity StudioからAPIを通じてフロントエンドに配信されます。
